# tac-website
static website for tech.at.core

___

## 🚀Prerequisites
* Node.js
* gatsby cli: `npm i -g gatsby-cli`
* typescript: `npm i -g gatsby-plugin-typescript`

## Setup
* clone this repo
* `cd tac-website`
* npm install


## How to run
* `gatsby develop`
